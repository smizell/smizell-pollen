The Verbose Hypermedia Format

17 May 2014

I’ve put up the first draft of a general-purpose, multi-use hypermedia format I called Verbose. Verbose is constructed from the main components of the Hyperextend library. Because of Verbose’s foundation, it should allow for covering any representation the other media types on Hyperschema.org can cover.

I’ve provided some uses and examples in the specs. It still has a ways to go, and if you have any thoughts or questions, don’t hesitate to get in touch or start a Github issue!
